# TypeScript 入门指南

TypeScript 是 JavaScript 的超集，它为 JS 添加了可选的静态类型和基于类的面向对象编程。

## 为什么选择 TypeScript？

1. 静态类型检查
2. IDE 智能提示
3. 面向对象特性
4. 更好的可维护性

## 快速上手

### 安装 TypeScript

```bash
npm install -g typescript
```

### 基础类型

```typescript
// 布尔值
let isDone: boolean = false;

// 数字
let decimal: number = 6;

// 字符串
let color: string = "blue";

// 数组
let list: number[] = [1, 2, 3];
```

## 进阶使用

- 接口
- 泛型
- 装饰器
- 类型推断
